# Navigator App

This project was bootstrapped with [create-navigator-app](https://github.com/fabriziosalmi/navigator).

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Next Steps

1. **Configure your app** - Edit `config.yaml` to customize layers and cards
2. **Add plugins** - Enable gesture, voice, or custom plugins
3. **Customize UI** - Modify `index.html` and add your own styles
4. **Deploy** - Build and deploy to any static hosting

## Keyboard Controls

- **Arrow Left / A** - Previous card
- **Arrow Right / D** - Next card
- **Arrow Up / W** - Layer up
- **Arrow Down / S** - Layer down
