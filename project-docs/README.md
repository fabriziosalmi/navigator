# Project Documentation

This directory contains **internal project documentation** that is not part of the public SDK documentation.

## Structure

- **`adrs/`** - Architectural Decision Records (ADRs)
- **`reports/`** - Sprint reports, audits, completion reports
- **`research/`** - Competitive analysis, future ideas, research notes

## Important

This directory is for **internal project management** only. 

For **public SDK documentation**, see:
- Source: `/apps/docs-site/`
- Published: https://navigator-docs.vercel.app (coming soon)

## Guidelines

### ADRs (Architectural Decision Records)

Document important architectural decisions:
- Why was this approach chosen?
- What alternatives were considered?
- What are the consequences?

### Reports

Sprint completion reports, audit results, migration reports.

### Research

Ideas for future features, competitive analysis, experimental designs.
